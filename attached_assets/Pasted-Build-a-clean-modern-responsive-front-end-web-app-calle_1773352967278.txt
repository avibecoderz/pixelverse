Build a clean, modern, responsive front-end web app called PixelStudio using React + Vite and Tailwind CSS.

Project type:
- Front-end only
- No backend yet
- Use dummy/sample data
- Focus on UI, layout, and user flow
- Make it look like a real SaaS dashboard for a photography management system

Brand:
- Name: PixelStudio
- Style: modern, minimal, professional
- Industry: photography studio management
- Use a clean dashboard design with cards, tables, forms, modals, and gallery sections

Roles:
Create two separate dashboards:
1. Admin Dashboard
2. Staff Dashboard

Main MVP idea:
This software helps photography studios manage client photo deliveries.
Staff can create a client record, upload photos, choose photo status, enter price, save the record, generate an invoice, and generate a shareable gallery link for the client.
Clients can open the gallery link and view/download their photos.
Admin can manage staff and see all payments received.
